在微信小程序中实现视频的播放与暂停

需求：

  • 视频列表中只能有一个视频在播放
  • 点击视频实现播放与暂停功能
  • 加载完成显示图片，点击后变为视频播放
  • 从上次播放的位置进行播放

思路：

  • 定义一个标记变量，控制视频的播放与暂停

    • true => 本次问播放
    • false => 本次为暂停

  • 每次点击后，更新data中视频的id值

  • 声明一个数组，用于存放播放视频的id和播放时间
